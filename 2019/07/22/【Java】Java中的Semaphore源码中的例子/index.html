Java中的Semaphore源码中的例子

Semaphore(信号量)是synchronized的加强版，作用是控制线程的并发数量。

信号量通过使用计数器来控制对共享资源的访问。如果计数器大于零，则允许访问。如果为零，则拒绝访问。

计数器的计数是允许访问共享资源的许可。因此，要访问资源，必须从信号量向线程授予许可。
