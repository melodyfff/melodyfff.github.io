单个项目的git配置

在git中，我们使用git config 命令用来配置git的配置文件，git配置级别主要有以下3类：

  • 1、仓库级别 local 【优先级最高】
  • 2、用户级别 global【优先级次之】
  • 3、系统级别 system【优先级最低】
